Toward Data Science Framework

Project Info:

This project is built to understand the temporal and geographical distribution of data science skills within the United State. More than 200K job descriptions were collected using cloud technologies and were analyzed by extracting skills and posted date and states.

The results were presented by deploying it over the internet using Dash (a web-based visualization framework)

Project Details:

  • Technologies:Cloud scrapying, scrapy Framework, AWS Mysql, AWS lambda/CloudWatch, Dash, Pandas
  • Date: 2021
  • Published Article: arXiv Version
