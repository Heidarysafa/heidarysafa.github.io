Railway Narratives Analysis with Deep Learning

Project Info:

Using deep learning on Railway accident narratives to predict the accident cause category. Both Conovolutional and recurrent Neural Networks have been considered while using two common word embedings.

Project Details:

  • Technologies:AWS EC2, RNN, CNN , Word2vec, Glove Embedding
  • Date:2018
  • Publ;ished Article:17th ICMLA
